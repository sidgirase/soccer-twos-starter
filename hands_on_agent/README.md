## Description

A starter agent module for the assignment. This folder contains the agent packaging required by `soccer_twos.watch` and serves as the base for the hands-on implementation.

## Usage

Use the agent with the project watcher:

```bash
python -m soccer_twos.watch -m hands_on_agent
```

If you train a model and save `checkpoint.pth` inside this folder, the agent will automatically load it on initialization.
